API Documentation

Universal Project Manager API
RESTful API для управления проектами с интеграцией AI
v3.1.0
🔑 Аутентификация
Для использования API необходимо получить API ключ. Добавьте его в заголовок Authorization.
GET /api/v3/introduction
Получение информации о API и доступных эндпоинтах.
cURL 
curl -X GET "https://api.universalpm.com/v3/introduction" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
200 OK
JSON 
{
  "success": true,
  "data": {
    "api_version": "3.1.0",
    "name": "Universal Project Manager API",
    "description": "RESTful API для управления проектами с интеграцией AI",
    "endpoints": [
      {
        "path": "/projects",
        "methods": ["GET", "POST"],
        "description": "Управление проектами"
      },
      {
        "path": "/tasks",
        "methods": ["GET", "POST", "PUT", "DELETE"],
        "description": "Управление задачами"
      }
    ]
  }
}
JavaScript
Использование с fetch API
const response = await fetch('https://api.universalpm.com/v3/introduction', { headers: { 'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json' } }); const data = await response.json();
Python
Использование с requests
import requests response = requests.get( 'https://api.universalpm.com/v3/introduction', headers={ 'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json' } ) data = response.json()
GET /api/v3/projects
Получение списка всех проектов пользователя с возможностью фильтрации и пагинации.
Параметр Тип Обязательный Описание
page integer Нет Номер страницы (по умолчанию: 1)
limit integer Нет Количество проектов на странице (по умолчанию: 20)
status string Нет Фильтр по статусу (active, completed, archived)
search string Нет Поиск по названию проекта
cURL 
curl -X GET "https://api.universalpm.com/v3/projects?page=1&limit=10&status=active" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
200 OK

  "data": {
    "projects": [
      {

}
POST /api/v3/projects
Создание нового проекта с указанием основных параметров.
Поле Тип Обязательное Описание
name string Да Название проекта
description string Нет Описание проекта
technologies array Нет Список используемых технологий
team_members array Нет Список участников команды
cURL 
curl -X POST "https://api.universalpm.com/v3/projects" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Vue.js Dashboard",
    "description": "Административная панель для управления системой",
    "technologies": ["Vue.js", "TypeScript", "Vite"],
    "team_members": ["user_123", "user_456"]
  }'
201 Created
JSON 
{
  "success": true,
  "data": {
    "id": "proj_456",
    "name": "Vue.js Dashboard",
    "description": "Административная панель для управления системой",
    "status": "active",
    "progress": 0,
    "created_at": "2025-01-31T15:30:00Z",
    "updated_at": "2025-01-31T15:30:00Z",
    "technologies": ["Vue.js", "TypeScript", "Vite"],
    "team_size": 2
  }
}
POST /api/v3/ai/analysis
Запуск AI-анализа проекта для получения рекомендаций по оптимизации.
Поле Тип Обязательное Описание
project_id string Да ID проекта для анализа
analysis_type string Нет Тип анализа (performance, security, quality)
ai_model string Нет AI модель для анализа (gpt4, claude, gemini)
cURL 
curl -X POST "https://api.universalpm.com/v3/ai/analysis" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "project_id": "proj_123",
    "analysis_type": "performance",
    "ai_model": "gpt4"
  }'
200 OK
JSON
